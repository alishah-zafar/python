# 🚨 Cyber-Attack Defense Terminal

A text-based cybersecurity terminal game where you play an engineer defending a live server from hackers — built in Python.

---

## 🎮 How to Play

You are a systems engineer. Your server is under attack. You have three options on every turn:

| Option | Action |
|--------|--------|
| `1` | Scan Network Logs — detect and block the hacker's IP |
| `2` | Deploy Firewall Override — solve a math puzzle under 5 seconds |
| `3` | Emergency Shutdown — end the session |

Your **server health starts at 100%**. Wrong decisions cost you 25% health. Reach 0% and it's game over.

---

## ▶️ Running the Game

**Requirements:** Python 3.x

```bash
python cyber_terminal.py
```

No external libraries needed — uses only Python's built-in `time` module.

---

## ⚙️ Concepts Used

- `while` loops for the continuous game engine
- `if/elif/else` conditional logic
- `time` module for real-time decision benchmarking
- f-strings for dynamic terminal output
- Type casting with `int()`
